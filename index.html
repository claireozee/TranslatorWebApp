About

This project was created in response to me wanting to learn French and my native language, Luganda. I decided to combine that interest with exploring the use of APIs in web applications, which resulted in me using the Google translate API on this app. The main aim was for me to replace my constant use of the Google translate interface, and replace it with something I've created instead, whilst still accessing the same content.
